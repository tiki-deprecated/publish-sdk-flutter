The Flutter implementation of TIKI's decentralized infrastructure plus abstractions to simplify the tokenization and application of data ownership, consent, and rewards. For new projects, we recommend one of our platform-specific SDKs. Same features. Much easier to implement.

It wraps the tiki-sdk-dart and adds specific implementations for Android and iOS.

This package should be used just for Flutter projects. For native implementation the specific SDKs are recommended.

Depend on it

Run this command with Flutter:

Now in your Dart code, you can use:
